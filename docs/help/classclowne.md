## First Time Usage

Using Class Clowne requires first having a [GitHub Account](https://github.com/signup)

When you click on a link to create your repository, you will encounter the sign-in page for the assignment

{% include image.html url="class-clowne-logged-out.png" description="Shows the assignment accept page" caption="Click sign in With GitHub" %}

When you click Sign in, you will be prompted to accept the permissions granted to Class Clowne to accept the invitation to be an outside collaborator on a repository.

{% include image.html url="class-clowne-authorize.png" description="Shows the authorize Class Clowne page" caption="Click Authorize" %}

The next assignment that you accept will not require authorization and will allow you to `Accept this assignment`

{% include image.html url="class-clowne-accept.png" description="Shows the link to create your repository with a button called Accept this assignment" caption="Click Accept this assignment" %}

After accepting the assignment, you will have access to your repository for the rest of the semester.

{% include image.html url="class-clowne-accepted.png" description="Shows the link to your personal repository" caption="Use the repository link to view your private repository" %}
